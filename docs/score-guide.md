# スコア管理 使い方

## 1. スコアを入力する

![スコア入力画面](./screenshots/score-scoring.png)

`/score` では、試合情報、スコア、選手登録を1画面で扱えます。

1. `大会名`、`試合タイトル`、`相手チーム` を入力します。
2. `年代タグ` と `日付` を設定します。
3. 必要なら `前後半`、`現在の区分`、`PK戦` を設定します。
4. 得点時は `ゴールを追加`、失点時は `失点を追加` を押します。
5. `得点選手` では、試合タグに合う登録選手だけが選択肢に出ます。

補足:
- スケジュール管理の試合から入ると、日付やタグなどが初期入力された状態で開けます。
- タイマーは `スタート / ストップ / リセット` で操作します。
- ゴールログは自動保存されます。

## 2. 選手を登録する

![選手登録画面](./screenshots/score-players.png)

1. `背番号`、`名前`、`グループ` を入力します。
2. `選手を追加` を押します。
3. 必要なら `試合結果から登録` で、既存試合の得点者からまとめて登録します。
4. 登録済み選手は `登録選手一覧` を開いて確認します。
5. 不要な選手は `削除` で消せます。

## 3. 試合結果一覧を見る

![試合結果一覧画面](./screenshots/score-results.png)

1. 上部タブで `試合結果一覧` を開きます。
2. `表示月` と `タグで絞り込み` で対象を絞ります。
3. `短縮 / 通常` を切り替えます。
4. `並び順` を変えると日付順、対戦相手順などで見られます。

補足:
- `短縮` では文字が小さくなり、`保存者 / 更新者` と `操作` 列が非表示になります。
- `通算結果`、`最多得点者ランキング`、`対戦相手別勝敗表` も同じ条件で再集計されます。

## 4. 試合結果を修正する

![試合修正画面](./screenshots/score-edit.png)

1. 試合結果一覧の `修正` を押します。
2. スコア入力画面へ戻るので、得点ログや得点選手を修正します。
3. 不要な得点は `得点を取り消す` で削除できます。
4. 更新しない場合は `修正を取り消す` を押します。
5. 反映するときは `試合結果を更新` を押します。

補足:
- 保存、更新、削除は LINE ログインが必要です。
- セッション切れの場合は再ログインに進みます。

## 5. CSV を使う

試合結果一覧には `CSVを書き出す` と `CSVを取り込む` があります。

補足:
- 書き出しは現在の絞り込み結果が対象です。
- 取り込みは既存の参考 CSV 形式に合わせています。
- 出力 CSV は Excel で開ける UTF-8 BOM 付きです。
