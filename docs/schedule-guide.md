# スケジュール管理 使い方

## 1. 月間予定を見る

![スケジュール管理トップ画面](./screenshots/schedule-top.png)

スマホで開くとトップはスケジュール管理です。`表示月`、`表示切替`、`学年` を選び、その下で月間予定を確認します。

補足:
- `短縮` は文字サイズを小さくし、できるだけ月全体が見える表示です。
- `通常` では `修正` と `操作` 列が出ます。
- `予定表を取り込む` は `日付 / 学年 / 時間 / 場所 / 内容` の CSV を読み込みます。

## 2. 出欠を入力する

![出欠入力モーダル](./screenshots/schedule-attendance-input.png)

1. 一覧の `出欠` を押します。
2. `参加 / 欠席 / 未定` を選びます。
3. 必要なら備考を入れます。
4. `出欠を保存` を押します。

補足:
- 出欠入力は LINE ログイン名で保存されます。
- セッション切れの場合は再ログインに進みます。

## 3. 出欠一覧を見る

![出欠一覧モーダル](./screenshots/schedule-attendance-list.png)

1. 一覧の `一覧` を押します。
2. 参加、欠席、未定の人数を確認します。
3. 各選手の出欠、入力者、更新日時、備考を確認します。

## 4. 当番を決める

![当番管理モーダル](./screenshots/schedule-duty.png)

1. 一覧の `当番` を押します。
2. `参加` にしている保護者の中から担当者を選びます。
3. 集合時間や役割メモを入力して `当番を保存` を押します。

補足:
- 決定者と決定日時は当番モーダル内で確認できます。
- 一覧表の当番欄には担当者名だけが出ます。

## 5. 予定を追加・修正する

![予定修正モーダル](./screenshots/schedule-edit.png)

1. 新規追加は `予定を追加`、修正は一覧の `修正` を押します。
2. `日付`、`タグ`、`場所`、`内容` を入力します。
3. 時間未定の場合は `開始` と `終了` に `-` を入力できます。
4. 試合なら `試合として扱う` をオンにします。
5. `予定を保存` または `予定を更新` を押します。

補足:
- 修正者は LINE ログイン名で記録されます。
- 試合として登録した予定は、スコア管理へ遷移したときの初期値に使われます。

## 6. Googleカレンダーへ取り込む

トップ画面の一番下に `Googleカレンダー取込` があります。

補足:
- 書き出し対象は、現在の `表示月` と `学年` の絞り込み結果です。
- Googleカレンダーへ取り込み後、このアプリとは同期されません。
