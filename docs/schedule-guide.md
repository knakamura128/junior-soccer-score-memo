# スケジュール管理 使い方

![スケジュール管理トップ画面](./screenshots/schedule-top.png)

スマホで開くと、トップはスケジュール管理画面です。上から `表示月`、`表示切替`、`学年` を選び、その下に月間予定一覧が表示されます。

## 1. 月間予定を見る

1. `表示月` で対象月を選びます。
2. 必要なら `学年` で絞り込みます。
3. 一覧をできるだけ多く見たいときは `短縮` を使います。

補足:
- `短縮` は文字サイズを小さくし、`修正` と `操作` の列を隠します。
- Googleカレンダー書き出しは、現在の `表示月` と `学年` の絞り込み結果が対象です。

## 2. 予定を追加する

1. 右上の `予定を追加` を押します。
2. `日付`、`タグ`、`場所`、`内容` を入力します。
3. 時間未定の場合は `開始` と `終了` に `-` を入力できます。
4. `予定を保存` を押します。

## 3. 予定を修正する

1. 一覧を `通常` 表示に切り替えます。
2. 対象行の `修正` を押します。
3. 内容を変更して `予定を更新` を押します。

補足:
- 修正、出欠、当番、削除は LINE ログインが必要です。
- セッション切れ時は再ログインに進むことがあります。

## 4. 出欠を入力する

1. 一覧を `通常` 表示に切り替えます。
2. 対象行の `出欠` を押します。
3. `参加 / 欠席 / 未定` を選び、必要なら備考を入力します。
4. `出欠を保存` を押します。

## 5. 出欠一覧を見る

1. 対象行の `一覧` を押します。
2. 入力者、出欠、備考、更新日時を確認します。

## 6. 当番を決める

1. 対象行の `当番` を押します。
2. `参加` にしている人の中から選択します。
3. `当番を保存` を押します。

補足:
- 誰が決めたかは当番モーダル内で確認できます。
- 一覧表の当番列は、確定した名前だけを表示します。

## 7. CSV を取り込む

1. 上部の `予定表を取り込む` から CSV を選びます。
2. 取り込み列は `日付 / 学年 / 時間 / 場所 / 内容` が基本です。

## 8. Googleカレンダーへ取り込む

1. ページ下部の `Googleカレンダー取込` を押します。
2. `.ics` ファイルが書き出されるので、Googleカレンダーへインポートします。

注意:
- 取り込み後もこのアプリと Googleカレンダーは同期されません。
- 現在の絞り込み結果だけが書き出されます。
